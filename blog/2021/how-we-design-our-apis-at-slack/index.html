5л…„ліҙлӢӨ м „м—җ, мҡ°лҰ¬лҠ” к°ңл°ңмһҗк°Җ мҠ¬лһҷм—җм„ң мүҪкІҢ м•ұмқ„ кө¬м¶•н•ҳкі  м•ұ л””л үн„°лҰ¬м—җ кІҢмӢңн•ҳлҠ” л°©лІ•мқ„ м ңкіөн•ҳлҠ” мҠ¬лһҷ н”Ңлһ«нҸјмқ„ лЎ м№ӯн–ҲмҠөлӢҲлӢӨ. мҳӨлҠҳлӮ , мҲҳл°ұл§Ң лӘ…мқҳ мӮ¬мҡ©мһҗк°Җ мҠ¬лһҷмңјлЎң мһ‘м—…мқ„ к°Җм ёмҳӨкі  мһҲмңјл©°, н”Ңлһ«нҸјм—җм„ң 885,000лӘ… мқҙмғҒмқҳ нҷңм„ұ к°ңл°ңмһҗк°Җ л§Ңл“  м•ұл“ӨмқҖ мҠ¬лһҷм—җм„ң нҳ‘м—…мқ„ лҚ”мҡұ н–ҘмғҒн•ҳкІҢ мӢңнӮӨлҠ” н•өмӢ¬мһ…лӢҲлӢӨ.

мҲҳл…„ лҸҷм•Ҳ мҡ°лҰ¬к°Җ к°ҖмһҘ мӨ‘мҡ”н•ҳкІҢ мғқк°Ғн•ҳлҠ” кІғмқҖ нӣҢлҘӯн•ң к°ңл°ңмһҗ кІҪн—ҳмқ„ м„Өкі„н•ҳлҠ” кІғмһ…лӢҲлӢӨ. лҲҲм№ҳмұ„м§Җ лӘ»н•ҳкІҢ кё°лҠҘ кө¬нҳ„мқ„ ліҖкІҪн•  мҲҳ мһҲм§Җл§Ң, кё°мЎҙ APIмқҳ лҸҷмһ‘ кі„м•Ҫмқ„ м ңкұ°н•ҳкұ°лӮҳ ліҖкІҪн•ҳлҠ” кІғмқҖ л§Өмҡ° м–ҙл өмҠөлӢҲлӢӨ. к·ёлҹ¬лҜҖлЎң API м„Өкі„лҘј мІҳмқҢл¶Җн„° мӢ мӨ‘н•ҳкІҢ мғқк°Ғн•ҳлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ.

APIк°Җ мһҳ м„Өкі„лҗңлӢӨл©ҙ, к°ңл°ңмһҗл“Өмқҙ мўӢм•„н•  кІғмқҙкі  лҳҗ к·ёл“ӨмқҖ APIлҘј мқҙмҡ©н•ҳм—¬ к°ҖмһҘ м°Ҫмқҳм Ғмқё нҳҒмӢ к°Җк°Җ лҗ  мҲҳлҸ„ мһҲмңјл©° лӮҳм•„к°Җ л§үлҢҖн•ң нҲ¬мһҗлҘј н•  кІғмқҙкі  л•ҢлЎңлҠ” APIмқҳ м „лҸ„мӮ¬к°Җ лҗ  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. мҡ°лҰ¬лҠ” лҳҗн•ң, н”Ңлһ«нҸјм—җм„ң к°ңл°ңмһҗмқҳ мӢңк°„кіј мң„н—ҳмқ„ к°җмҲҳн•ҳлҠ” мһҗмӣҗ(resource)мқ„ мҶҢмӨ‘н•ҳкІҢ мғқк°Ғн•©лӢҲлӢӨ. мһҳлӘ»лҗң API м„Өкі„лҠ” мөңмҶҢн•ңмңјлЎң мұ„нғқлҗ  кІғмқҙл©° мӢ¬м§Җм–ҙ мўҢм Ҳк°җмқ„ мҙҲлһҳн•©лӢҲлӢӨ. мһҳлӘ»лҗң APIлҠ” кё°м—…мқҳ мұ…мһ„мқҙ лҗ©лӢҲлӢӨ.

к·ёл ҮлӢӨкі  мҠ¬лһҷмқҙ н•ӯмғҒ APIлҘј мһҳ м„Өкі„н•ҙ мҷ”лӢӨкі  л§җн•ҳлҠ” кІғмқҖ м•„лӢҷлӢҲлӢӨ. мҡ°лҰ¬к°Җ мӢӨмҲҳлҘј м Җм§ҖлҘёлӢӨл©ҙ, н”Ңлһ«нҸјмқҖ ліҙлӢӨ к°ңл°ңмһҗ м№ңнҷ”м Ғ кІҪн—ҳмқ„ м ңкіөн•  мҲҳ мһҲмқ„ кІғмһ…лӢҲлӢӨ. мқҙлҹ¬н•ң мӢӨмҲҳлҘј мқёмӢқн•ҳкі  к°ңм„ н•ҳлҠ” л°©лІ•мқ„ нҢҢм•…н•ЁмңјлЎңмҚЁ - мӢ¬м§Җм–ҙ кіјкұ°мқҳ м„ нғқкіј мқјкҙҖм„ұмқ„ лҒҲм§Ҳкё°кІҢ мң м§Җн•ҳлҠ” кІғм—җ лҢҖн•ҙ нҳ„мһ¬ мҡ°лҰ¬к°Җ лҸҷмқҳн•ҳм§Җ м•Ҡмқ„ л•ҢлҸ„ мһҲмҠөлӢҲлӢӨ - к°ңл°ңмһҗ кІҪн—ҳмқ„ м „мІҙм ҒмңјлЎң к°ңм„ н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мһҗмӢ л§Ңмқҳ API мҠӨнғҖмқј к°Җмқҙл“ңлҘј к°ңл°ңн•ңлӢӨкі  н•ҙм„ң нҳ•нҺём—ҶлҠ” кІ°м •мқ„ лӮҙлҰ¬кұ°лӮҳ мҳӨлҠҳмқҳ м—ҙм •м Ғмқё м„ нғқмқҙ лӮҙмқјмқҳ нӣ„нҡҢк°Җ лҗҳлҠ” кІғмқ„ мҷ„м „нһҲ л§үмқ„ мҲҳлҠ” м—ҶмҠөлӢҲлӢӨ. к·ёл Үм§Җл§Ң мҶ”м§Ғн•ҳкІҢ, м •м§Ғн•ҳкІҢ, лӘ…нҷ•н•ҳкІҢ кІ°м •мқ„ лӮҙлҰ¬лҠ” лҚ° лҸ„мӣҖмқҙ лҗ  кІғмһ…лӢҲлӢӨ. мқҙ кёҖм—җм„  мҡ°лҰ¬мқҳ API м„Өкі„ мӣҗм№ҷкіј лҚ”л¶Ҳм–ҙ мғҲлЎңмҡҙ API мӮ¬м–‘, кІҖнҶ  л°Ҹ н…ҢмҠӨнҠён•ҳлҠ” н”„лЎңм„ёмҠӨм—җ кҙҖн•ҙ м„ӨлӘ…н•©лӢҲлӢӨ. л§Ҳм§Җл§үм—җлҠ”, м—¬лҹ¬л¶„мқҳ API н”„лЎңм„ёмҠӨлЎң к°Җм§Җкі  к°Ҳ мҲҳ мһҲлҠ” лӘҮ к°Җм§Җ м•„мқҙл””м–ҙлҘј м–»мқ„ мҲҳ мһҲмқ„ кІғмһ…лӢҲлӢӨ.

м„Өкі„ мӣҗм№ҷ

мҲҳл…„м—җ кұёміҗ, API м„Өкі„лҘј к°Җмқҙл“ңн•ҳлҠ” лӘҮ к°Җм§Җ мӣҗм№ҷмқ„ м •н–ҲмҠөлӢҲлӢӨ.

1. н•ң к°Җм§Җ мқјмқ„ мһҳ н•ҙлӮҙлқј

API м„Өкі„лҘј мӢңмһ‘н•ҳл©ҙ н•ң лІҲм—җ л„Ҳл¬ҙ л§ҺмқҖ л¬ём ңлҘј н•ҙкІ°н•ҳл Өкі  н•ҳлҠ” мң нҳ№мқҙ мғқк№ҒлӢҲлӢӨ. л§ҺмқҖ кІғмқ„ н•ҳл Өкі  н• мҲҳлЎқ, ліөмһЎн•ҙм§Җкі  мқҙн•ҙн•ҳкё° м–ҙл ӨмӣҢм§‘лӢҲлӢӨ. нҠ№м • мӮ¬мҡ© мӮ¬лЎҖлҘј нғқн•ҳл©ҙ лӢЁмқј м„Өкі„м—җ м§‘мӨ‘н•ҳкі  APIлҘј лӢЁмҲңн•ҳкІҢ мң м§Җн•  мҲҳ мһҲмҠөлӢҲлӢӨ. лӢЁмҲңн•ң APIлҠ” мқҙн•ҙн•ҳкё° мү¬мҡё лҝҗл§Ң м•„лӢҲлқј нҷ•мһҘ, м„ұлҠҘ н–ҘмғҒ л°Ҹ м•Ҳм „м„ұлҸ„ н–ҘмғҒлҗ©лӢҲлӢӨ. лҳҗн•ң APIм—җ мғҲлЎңмҡҙ кё°лҠҘмқ„ м¶”к°Җн•ҳкё°лҠ” мүҪм§Җл§Ң м ңкұ°н•ҳкё°лҠ” м–ҙл өмҠөлӢҲлӢӨ.

мҠ¬лһҷм—җм„ңлҠ” лҢҖл¶Җл¶„мқҳ APIк°Җ мқҙлҹ¬н•ң мІ н•ҷмқ„ л”°лҘҙкі  мһҲмҠөлӢҲлӢӨ. к·ёлҹ¬лӮҳ м–ҙл–Ө кІҪмҡ°м—җлҠ” кё°мЎҙмқҳ ліөмһЎн•ң APIлҘј ліҙлӢӨ лӢЁмҲңн•ҳкі  м§‘мӨ‘м Ғмқё APIлЎң лӮҳлҲ м•ј н–ҲмҠөлӢҲлӢӨ. к°ҖмһҘ мқёкё° мһҲлҠ” API л©”м„ңл“ң мӨ‘ н•ҳлӮҳмқё rtm.startлҠ” мӢңк°„мқҙ м§ҖлӮҳл©ҙм„ң мғҒлӢ№нһҲ л№„мҡ©мқҙ л№„мӢёмЎҢмҠөлӢҲлӢӨ. мқҙ л©”м„ңл“ңлҠ” нҢҖ, мұ„л„җ л°Ҹ л©ӨлІ„м—җ лҢҖн•ң лӢӨм–‘н•ң лҚ°мқҙн„°мҷҖ н•Ёк»ҳ мӣ№ мҶҢмј“ APIм—җ м—°кІ°н•ҳкё° мң„н•ң URLмқ„ л°ҳнҷҳн•©лӢҲлӢӨ. нҢҖ к·ңлӘЁк°Җ м»Өм§җм—җ л”°лқј мқҙ нҺҳмқҙлЎңл“ңлҠ” лӢӨлЈЁкё° м–ҙл өкі  м»Өм ём„ң к°ңл°ңмһҗл“Өмқҙ к°җлӢ№н•ҳкё°м—җлҠ” л№„мҡ©мқҙ л§Һмқҙ л“Өм—ҲмҠөлӢҲлӢӨ.

мҶҢмҲҳмқҳ к°ңл°ңмһҗк°Җ мқҙ л©”м„ңл“ңм—җм„ң л°ҳнҷҳлҗң лҚ°мқҙн„°лҘј мӮ¬мҡ©н–Ҳм§Җл§Ң, к°ңл°ңмһҗ лҢҖл¶Җл¶„мқҖ мӣ№ мҶҢмј“м—җл§Ң м—°кІ°н•ҳкё°лҘј мӣҗн–ҲмҠөлӢҲлӢӨ. к·ё кІ°кіј, мҠ¬лһҷмқҖ нҺҳмқҙлЎңл“ңм—җ лӢӨлҘё лҚ°мқҙн„°лҘј л°ҳнҷҳн•ҳм§Җ м•Ҡкі  мӣ№ мҶҢмј“ API м„ём…ҳ URLмқ„ л°ҳнҷҳн•ҳлҠ” мғҲлЎңмҡҙ API л©”м„ңл“ңмқё rtm.connectлҘј лҸ„мһ…н–ҲмҠөлӢҲлӢӨ. мқҙ мғҲлЎңмҡҙ л©”м„ңл“ңлҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳ к°ңл°ңмһҗмҷҖ мҠ¬лһҷмқҙ rtm.startмқҳ нҷ•мһҘ л¬ём ңлҘј н•ҙкІ°н•ҳлҠ” лҚ° лҸ„мӣҖмқҙ лҗҳм—ҲмҠөлӢҲлӢӨ.

мқҙ кө¬мІҙм Ғмқё мқҙм•јкё°м—җм„ң м–»мқ„ мҲҳ мһҲлҠ” н•ң к°Җм§ҖлҠ”: нҷ•мӢӨн•ҳм§Җ м•Ҡмқ„ л•Ң, м–ҙл–Ө м»¬л үм…ҳм—җм„ңл“  н•ңм •лҗң мҲҳмқҳ к°қмІҙлҘј м Ғмҡ©н•ҳкұ°лӮҳ нҺҳмқҙм§Җнҷ”лҘј н•ҳлҠ” кІғмһ…лӢҲлӢӨ. м •мғҒм Ғмқҙкі  н•©лҰ¬м Ғмқё мғҒн•ңмқ„ м •мқҳн•ҳлҠ” кІғмқҖ мӢңкё°мғҒмЎ°мқё мөңм Ғнҷ”к°Җ м•„лӢҷлӢҲлӢӨ. мң кё°м Ғ м„ұмһҘмқҙ к·ё кІҪкі„к°Җ м–ҙл””м—җ мһҲлҠ”м§Җ ліҙм—¬мӨ„ л•Ң, лҚ” н•©лҰ¬м Ғмқё кІғмқ„ нҸ¬н•Ён•ҳкё°к°Җ нӣЁм”¬ лҚ” м–ҙл өмҠөлӢҲлӢӨ.

2. л№ лҘҙкі  мүҪкІҢ мӢңмһ‘н•ҳлқј

к°ңл°ңмһҗк°Җ APIлҘј мқҙн•ҙн•ҳкі  мӢ мҶҚн•ҳкІҢ мӢңмһ‘н•  мҲҳ мһҲлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ. APIм—җ мқөмҲҷн•ҳм§Җ м•ҠмқҖ к°ңл°ңмһҗк°Җ м Ғм Ҳн•ң мӢңк°„ лӮҙм—җ вҖңHello worldвҖқ м—°мҠөмқ„ мҷ„лЈҢн•  мҲҳ мһҲлҸ„лЎқ ліҙмһҘн•ҙм•ј н•©лӢҲлӢӨ. мқҙлҘј нҸүк°Җн•ҳлҠ” лҚ° мӮ¬мҡ©н•  мҲҳ мһҲлҠ” л©”нҠёлҰӯ мӨ‘ н•ҳлӮҳк°Җ вҖңмІ« Hello Worldк№Ңм§Җмқҳ мӢңк°„вҖқмһ…лӢҲлӢӨ. мҠ¬лһҷм—җм„ңлҠ” мҙҲкёү к°ңл°ңмһҗк°Җ н”Ңлһ«нҸјм—җ лҢҖн•ҙ л°°мҡ°кі , м•ұмқ„ л§Ңл“Өкі , м•Ҫ 15л¶„ мқҙлӮҙм—җ мІ« API нҳём¶ңмқ„ ліҙлӮј мҲҳ мһҲкё°лҘј л°”лһҚлӢҲлӢӨ. лӘ©н‘ңмқё вҖңмІ« Hello Worldк№Ңм§Җмқҳ мӢңк°„вҖқмқҖ мӢӨн–үн•ҳлҠ” н”Ңлһ«нҸјкіј к°ңл°ңмһҗ лҢҖмғҒмқҳ кІҪн—ҳ мҲҳмӨҖм—җ л”°лқј лӢ¬лқјм§‘лӢҲлӢӨ.

мӢңмһ‘н•ҳкё° к°Җмқҙл“ң, мһҗмҠөм„ң, мғҳн”Ң мҪ”л“ң л°Ҹ мғҒнҳёмһ‘мҡ©мқ„ н•ҳлҠ” л¬ём„ңлҠ” к°ңл°ңмһҗк°Җ л№ лҘҙкІҢ мӢңмһ‘н•  мҲҳ мһҲлҸ„лЎқ л§ҺмқҖ лҸ„мӣҖмқ„ мӨҚлӢҲлӢӨ. мҠ¬лһҷм—җм„ , мӢңмһ‘н•ҳкё° к°Җмқҙл“ңмҷҖ мһҗмҠөм„ң мҷём—җлҸ„ к°ңл°ңмһҗк°Җ лёҢлқјмҡ°м Җм—җм„ң API м—”л“ңнҸ¬мқёнҠёлҘј мӮ¬мҡ©н•ҙ ліј мҲҳ мһҲлҠ” мғҒнҳёмһ‘мҡ©мқ„ н•ҳлҠ” API н…ҢмҠӨн„°к°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ. лҳҗн•ң м җм җ лҠҳм–ҙлӮҳлҠ” SDK м»¬л үм…ҳмқ„ мӮ¬мҡ©н•ҳм—¬ мҪ”л“ңм—җ мүҪкІҢ м—°кІ°н•  мҲҳ мһҲлҠ” м—¬лҹ¬ м–ём–ҙлЎң лҗң мҪ”л“ң мЎ°к°Ғмқ„ м§ңкё° мӢңмһ‘н–ҲмҠөлӢҲлӢӨ.

3. м§ҒкҙҖм Ғмқё мқјкҙҖм„ұмқ„ мң„н•ҙ л…ёл Ҙн•ҳлқј

APIк°Җ м§ҒкҙҖм ҒмңјлЎң мқјкҙҖм„ұмқ„ мң м§Җн•ҙм•ј н•©лӢҲлӢӨ. мқҙлҠ” м—”л“ңнҸ¬мқёнҠё мқҙлҰ„, мһ…л Ҙ л§Өк°ң ліҖмҲҳ(input parameters) л°Ҹ м¶ңл Ҙ мқ‘лӢө(output responses)м—җ л°ҳмҳҒлҗҳм–ҙм•ј н•©лӢҲлӢӨ. к·ёлҰ¬кі  к°ңл°ңмһҗлҠ” м„ӨлӘ…м„ңлҘј мқҪм§Җ м•Ҡм•„лҸ„ API мқјл¶ҖлҘј м¶”мёЎн•  мҲҳ мһҲм–ҙм•ј н•©лӢҲлӢӨ. лҳҗн•ң к·ёл“Өмқҙ мқҙлҜё м•Ңкі  мһҲлҠ” кІғкіј мң мӮ¬н•ҳкІҢ мһ‘лҸҷн•ҙм•ј н•©лӢҲлӢӨ. м•„лһҳлҠ” 3к°Җм§Җ мҲҳмӨҖмқҳ мқјкҙҖм„ұмһ…лӢҲлӢӨ:

  • м—…кі„ н‘ңмӨҖкіјмқҳ мқјкҙҖм„ұ: м—…кі„м—җм„ң л°ӣм•„л“ӨмқҙлҠ” лӘЁлІ” мӮ¬лЎҖлҘј мөңлҢҖн•ң мӨҖмҲҳн•ҙм•ј н•©лӢҲлӢӨ.
  • м ңн’Ҳкіјмқҳ мқјкҙҖм„ұ: м ңн’Ҳм—җм„ң мқҙлҹ¬н•ң к°ңл…җмқ„ кё°л°ҳмңјлЎң н•„л“ң мқҙлҰ„мқ„ м„ нғқн•ҙм•ј н•©лӢҲлӢӨ. м•Ҫм–ҙ, л‘җл¬ёмһҗм–ҙ к·ёлҰ¬кі  м „л¬ё мҡ©м–ҙлҠ” н”јн•ҳмӢӯмӢңмҳӨ. мһҘнҷ©н•ҳкІҢ л§җм”Җн•ҳмӢӯмӢңмҳӨ.
  • лӢӨлҘё API л©”м„ңл“ңмҷҖмқҳ мқјкҙҖм„ұ: лӢӨлҘё API л©”м„ңл“ңм—җм„ң мӮ¬мҡ©н•ҳлҠ” мқҙлҰ„мқҖ м„ңлЎң мқјм№ҳн•ҙм•ј н•©лӢҲлӢӨ.

API м„Өкі„м—җ лҢҖн•ң мһҗмӢ мқҳ мқҳкІ¬мқ„ кіөкі нһҲ н•ҳкі  кё°лЎқн•ҳлҠ” кІғмқҙ мқјкҙҖм„ұмқ„ нҷ•ліҙн•ҳлҠ” к°ҖмһҘ мўӢмқҖ л°©лІ•мқҳ н•ҳлӮҳмһ…лӢҲлӢӨ. нҠ№нһҲ API м„Өкі„м—җ лҢҖн•ҙ мҳікі  к·ёлҘё л°©лІ•мқҙ м—Ҷмқ„ л•ҢлҠ” лҚ”мҡұ к·ёл ҮмҠөлӢҲлӢӨ. м„ нғқн•ҳкі  к·ёкІғмқ„ кі мҲҳн•ҳмӢӯмӢңмҳӨ. мҠ¬лһҷм—җм„ңлҠ” APIм—җ лҢҖн•ң мқјкҙҖлҗң кҙҖн–үкіј нҢЁн„ҙмқ„ м •мқҳн•ҳлҠ” нҸ¬кҙ„м Ғмқё API м„Өкі„ к°Җмқҙл“ңлқјмқёмқ„ мһ‘м„ұн–ҲмҠөлӢҲлӢӨ.

4. мқҳлҜёмһҲлҠ” м—җлҹ¬лҘј л°ҳнҷҳн•ҳлқј

APIлҘј м„Өкі„н•ҳлҠ” лҳҗ лӢӨлҘё мӣҗм№ҷмқҖ мқҳлҜё мһҲлҠ” м—җлҹ¬лҘј л°ҳнҷҳн•ҳм—¬ к°ңл°ңмһҗк°Җ л¬ём ңлҘј лҚ”мҡұ мүҪкІҢ н•ҙкІ°н•  мҲҳ мһҲлҸ„лЎқ н•ҳлҠ” кІғмһ…лӢҲлӢӨ. API м„Өкі„ мӨ‘м—җ м—”м§ҖлӢҲм–ҙлҠ” APIм—җ мқҳн•ҙ л°ҳнҷҳлҗҳлҠ” м—җлҹ¬м—җ кұ°мқҳ мЈјмқҳлҘј кё°мҡёмқҙм§Җ м•ҠмҠөлӢҲлӢӨ. л¶Җм •нҷ•н•ҳкұ°лӮҳ л¶Ҳл¶„лӘ…н•ң м—җлҹ¬лҠ” л¶Ҳл§ҢмҠӨлҹ¬мҡё мҲҳ мһҲкі  API мұ„нғқм—җ л¶Җм •м Ғмқё мҳҒн–Ҙмқ„ лҜём№  мҲҳ мһҲмҠөлӢҲлӢӨ. к·ёлҰ¬н•ҳм—¬ к°ңл°ңмһҗлҠ” л§үнҳҖм„ң нҸ¬кё°н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мўӢмқҖ м—җлҹ¬лҠ” мқҙн•ҙн•ҳкё° мүҪкі  лӘЁнҳён•ҳм§Җ м•Ҡмңјл©° мӢӨн–ү к°ҖлҠҘн•ҳл©° л¬ём ңлҘј мқҙн•ҙн•ҳкі  м–ҙл–»кІҢ н•ҙкІ°н• м§Җ лҸ„мҡё кІғмһ…лӢҲлӢӨ. APIм—җм„ң кө¬нҳ„ м„ёл¶Җ м •ліҙ, нҠ№нһҲ м—җлҹ¬к°Җ л№ м ём„ңлҠ” м•Ҳ лҗ©лӢҲлӢӨ.

м—җлҹ¬ мҪ”л“ңлҘј л°ҳнҷҳн•ҳлҠ” кІғ мҷём—җлҸ„ м„ӨлӘ…м„ңлӮҳ API мқ‘лӢөмқҳ лӢӨлҘё кіім—җм„ң кёҙ нҳ•мӢқмқҳ м—җлҹ¬лҘј м¶”к°Җн•ҳлҠ” кІғмқҙ мў…мў… мң мҡ©н•©лӢҲлӢӨ. м—¬кё°м—җлҠ” л¬ём ң н•ҙкІ° л°©лІ•мқҙлӮҳ м¶”к°Җ м •ліҙ л§ҒнҒ¬лҘј нҸ¬н•Ён•ҳм—¬ мӮ¬мҡ©мһҗк°Җ мқҪмқ„ мҲҳ мһҲлҠ” м—җлҹ¬ м„ӨлӘ…мқҙ нҸ¬н•Ёлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

к°ңл°ңмһҗлҘј мң„н•ң SDK л°Ҹ лқјмқҙлёҢлҹ¬лҰ¬лҸ„ кө¬м¶•н•ҳлҠ” кІҪмҡ° м—җлҹ¬ л©”мӢңм§Җ л°Ҹ кІҪкі лҘј вҖңмӮјнӮӨм§ҖвҖқ м•ҠлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ. к°ңл°ңмһҗк°Җ HTTP н—ӨлҚ” л°Ҹ мӣҗмӢң(raw) мҡ”мІӯ ліёл¬ё(body)мҷҖ к°ҷмқҙ лҒ”м°Қн•ң л””лІ„к№… м„ём…ҳм—җм„ң мң мҡ©н•  мҲҳ мһҲлҠ” лӘЁл“  кІғм—җ м•Ўм„ёмҠӨн•  мҲҳ мһҲлҠ”м§Җ нҷ•мқён•ҳмӢӯмӢңмҳӨ. SDKк°Җ м—җлҹ¬лҘј н•ҙм„қн•ҳм—¬ нӣЁм”¬ лҚ” мң мҡ©н•ҳкІҢ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨл©ҙ мўӢкІ м§Җл§Ң, к°ңл°ңмһҗлҠ” м—җлҹ¬м—җ лҢҖн•ң мӣҗмӢң(raw) API л¬ём„ңлҘј мқҪкі  SDK мҲҳмӨҖм—җм„ң м—җлҹ¬лҘј м •нҷ•нһҲ нҢҢм•…н•  мҲҳ мһҲм–ҙм•ј н•©лӢҲлӢӨ. мҡ°лҰ¬лҠ” л•Ңл•ҢлЎң мҡ°лҰ¬ н”Ңлһ«нҸјмқҳ мң мҡ©н•ң м—җлҹ¬ л©”мӢңм§ҖлҘј к°ңл°ңмһҗл“Өм—җкІҢ лҸ„л‘‘м§Ҳн•ҳм—¬ мЈ„лҘј м§Җкё°лҸ„ н•©лӢҲлӢӨ.

5. к·ңлӘЁмҷҖ м„ұлҠҘмқ„ мң„н•ң м„Өкі„лҘј н•ҳлқј

мһҳлӘ»лҗң м„Өкі„лҠ” м„ұлҠҘмқ„ м ңн•ңн•  мҲҳ мһҲмҠөлӢҲлӢӨ. к·ёлһҳм„ң APIлҘј м„Өкі„н•  л•Ң лӢӨмқҢкіј к°ҷмқҖ лӘҮ к°Җм§Җ мӮ¬н•ӯмқ„ нҷ•мқён•ҙм•ј н•©лӢҲлӢӨ:

  • нҒ° м»¬л үм…ҳмқ„ нҺҳмқҙм§Җнҷ” н•ҳмӢӯмӢңмҳӨ: APIлҠ” лҢҖк·ңлӘЁ лҚ°мқҙн„° м„ёнҠёлҘј мІҳлҰ¬н•ҙм•ј н•ҳлҠ” кІҪмҡ°к°Җ м•„мЈј л§ҺмҠөлӢҲлӢӨ. API нҳём¶ң н•ң лІҲмңјлЎң мҲҳмІң к°ңмқҳ н•ӯлӘ©мқҙ л°ҳнҷҳлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлЎң мқён•ҙ мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳ л°ұм—”л“ңм—җ кіјл¶Җн•ҳк°Җ кұёлҰҙ мҲҳ мһҲмңјл©° мӢ¬м§Җм–ҙ лҢҖк·ңлӘЁ лҚ°мқҙн„° м„ёнҠёлҘј мІҳлҰ¬н•  мҲҳ м—ҶкІҢ нҒҙлқјмқҙм–ёнҠёк°Җ лҠҗл Өм§Ҳ мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҹ° мқҙмң лЎң, нҒ° кІ°кіј м„ёнҠёлҘј нҺҳмқҙм§Җнҷ” н•ҳлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ.
  • нҒ° м»¬л үм…ҳ м•Ҳм—җ лӢӨлҘё нҒ° м»¬л үм…ҳм—җ мӨ‘мІ©н•ҳм§Җ л§ҲмӢӯмӢңмҳӨ: мқҙ кІҪмҡ° нҺҳмқҙм§Җнҷ”лҠ” л„Ҳл¬ҙ ліөмһЎн•©лӢҲлӢӨ.
  • API л№„мңЁмқ„ м ңн•ңн•ҳмӢӯмӢңмҳӨ: м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ лӢӨмҡҙмӢңнӮӨлҠ” мһҳлӘ» н–үлҸҷн•ҳлҠ” н•ң лӘ…мқҳ к°ңл°ңмһҗлӮҳ м ңм–ҙк°Җ м•Ҳ лҗҳлҠ” мҪ”л“ң лЈЁн”„лҘј мӣҗн•ҳм§Җ м•Ҡмқ„ кІғмһ…лӢҲлӢӨ. мқён”„лқјлҘј ліҙнҳён•ҳлҠ” лҸҷмӢңм—җ м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ м•Ҳм •м„ұкіј к°Җмҡ©м„ұмқ„ лҶ’мқҙл Өл©ҙ н•©лҰ¬м Ғмқё л№„мңЁ м ңн•ңмқ„ м¶”к°Җн•ҙм•ј н•©лӢҲлӢӨ.

мҠ¬лһҷмқҳ API мӨ‘ н•ҳлӮҳмқё channels.list(м§ҖкёҲмқҖ мӮ¬мҡ© мӨ‘м§Җ)лҠ” мұ„л„җ лӘ©лЎқкіј к°Ғ мұ„л„җмқҳ лӘЁл“  л©ӨлІ„лҘј л°ҳнҷҳн•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳм—ҲмҠөлӢҲлӢӨ. вҖ” кё°ліём ҒмңјлЎң м»¬л үм…ҳмқ„ лӢӨлҘё м»¬л үм…ҳм—җ мӨ‘мІ©н•©лӢҲлӢӨ. мҠ¬лһҷмқҙ л§Ңл“Өм–ҙмЎҢмқ„ л•Ң, мҡ°лҰ¬лҠ” к°Ғ нҢҖмқҙ мҲҳл°ұ лӘ…мқҳ мӮ¬мҡ©мһҗлЎң м ңн•ңлҗ  кІғмқҙлқјлҠ” к°Җм •мқ„ н–ҲмҠөлӢҲлӢӨ. к·ёлҹ¬лӮҳ м ңн’Ҳмқҙ л°ңм „н•ҳкі  м„ұмһҘн•Ём—җ л”°лқј мқҙлҹ¬н•ң к°Җм •мқҖ лҚ”лҠ” мӮ¬мӢӨмқҙ м•„лӢҲкІҢ лҗҳм—ҲмҠөлӢҲлӢӨ. мҡ°лҰ¬лҠ” мҲҳл§Ң лӘ…мқҳ мӮ¬мҡ©мһҗлҘј к°Җм§„ нҢҖмқ„ кө¬м„ұн•ҳкё° мӢңмһ‘н–ҲмҠөлӢҲлӢӨ. к·ё мӢңм җм—җ, к°ҷмқҖ API нҳём¶ңм—җм„ң лӘЁл“  мұ„л„җкіј к°Ғ мұ„л„җмқҳ л©ӨлІ„лҘј л°ҳнҷҳн•ҙм•ј н•ҳлҠ” мқҙ кө¬мЎ°лҠ” м§Җмӣҗн•ҳкё°к°Җ л„Ҳл¬ҙ м–ҙл ӨмӣҢмЎҢмҠөлӢҲлӢӨ. к·ёлһҳм„ң к·ёлӢӨмқҢ мқҙлҘј л‘җ к°ңмқҳ м„ңлЎң лӢӨлҘё к°„лӢЁн•ң APIлЎң лӮҳлҲҙмҠөлӢҲлӢӨ: conversations.listмҷҖ conversations.members. мқҙлҠ” API м„ұлҠҘмқ„ н–ҘмғҒн•  лҝҗл§Ң м•„лӢҲлқј к°ңл°ңмһҗ кІҪн—ҳ н–ҘмғҒм—җлҸ„ лҸ„мӣҖмқҙ лҗҳм—ҲмҠөлӢҲлӢӨ.

нҺҳмқҙм§Җнҷ”лҠ” мӨ‘мҡ”н•  лҝҗл§Ң м•„лӢҲлқј к·ё мӢӨмҲҳмқҳ нҢЁн„ҙмқҙ к°ҷмңјлҜҖлЎң нҺҳмқҙм§Җнҷ”м—җ лҢҖн•ң мЈјм ңлҘј лӢӨмӢң м–ёкёүн•©лӢҲлӢӨ. м—¬лҹ¬л¶„м—җкІҢ вҖңк·ё м •лҸ„л©ҙ м¶©л¶„н•ҳлӢӨвҖқлқјлҠ” л§җмқҖ м•„л§Ҳ лӢӨлҘё мӮ¬лһҢл“Өм—җкІҢ м¶©л¶„н•ҳм§Җ м•Ҡмқ„ кІҒлӢҲлӢӨ. APIмқҳ мҰҗкұ°мӣҖ мӨ‘ н•ҳлӮҳлҠ” лӢӨлҘё к°ңл°ңмһҗк°Җ APIлҘј м–ҙл–»кІҢ мӮ¬мҡ©н• м§Җ м „нҳҖ лӘЁлҘёлӢӨлҠ” кІғмһ…лӢҲлӢӨ. к·ёлҹ¬лӮҳ нҠ№нһҲ к°ҷмқҖ APIлҘј мӮ¬мҡ©н•ҳлҠ” мӮ¬мҡ©мһҗлқјл©ҙ к·ёл“Өмқ„ ліј мҲҳ м—ҶкІҢ н•ҳлҠ” л§ҺмқҖ к°ҖлҠҘм„ұмқ„ мҳҲмғҒн•ҳлҠ” кІғмқҖ м—¬м „нһҲ м—¬лҹ¬л¶„м—җкІҢ лӢ¬л Ө мһҲмҠөлӢҲлӢӨ.

л§Ҳм§Җл§үмңјлЎң м ңн’Ҳмқҳ мғҲ м—…лҚ°мқҙнҠёлҘј м¶ңмӢңн•  л•Ң API ліҖкІҪмңјлЎң мқён•ҙ нҒҙлқјмқҙм–ёнҠёмқҳ мҪ”л“ңлҘј к№ЁлҠ” мқјмқҖ м ҲлҢҖ л°ңмғқн•ҳм§Җ м•ҠкІҢ н•ҳлҠ” кІғмһ…лӢҲлӢӨ. к·ёлһҳм„ң, нҳёнҷҳм„ұмқ„ к№ЁлҠ” ліҖкІҪ мӮ¬н•ӯмқҖ л¬ҙм—Үмһ…лӢҲк№Ң? кё°мЎҙ м•ұмқҙ ліҖкІҪ м „мІҳлҹј мһ‘лҸҷн•ҳм§Җ м•Ҡмқ„ мҲҳ мһҲлҠ” лӘЁл“  ліҖкІҪ мӮ¬н•ӯмһ…лӢҲлӢӨ.

мҠ¬лһҷм—җм„ң мҡ°лҰ¬к°Җ л”°лҘҙлҠ” мІ н•ҷмқҖ м–ҙм ң лҸҷмһ‘н–ҲлҚҳ кІғмқҙ лӮҙмқјлҸ„ лҸҷмһ‘н•ҙм•ј н•ңлӢӨлҠ” кІғмһ…лӢҲлӢӨ. к·ёл Үкёҙ н•ҳм§Җл§Ң, ліҖнҷ”лҠ” л¶Ҳк°Җн”јн•©лӢҲлӢӨ. л“ңл¬јм§Җл§Ң мҳҲмҷём Ғмқё кІҪмҡ°, мҡ°лҰ¬лҠ” нҳёнҷҳм„ұмқ„ к№ЁлҠ” ліҖкІҪ мӮ¬н•ӯмқ„ лҸ„мһ…н•ҳкё°лЎң н•©лӢҲлӢӨ. к°ңл°ңмһҗмқҳ мӣҗнҷңн•ң кІҪн—ҳмқ„ мң„н•ҙ м–јл§ҲлӮҳ л§ҺмқҖ кіөм§ҖмӮ¬н•ӯмқ„ м „лӢ¬н•ҳкі  м–ҙлҠҗ м •лҸ„к№Ңм§Җ м§„н–үн•  кІғмқём§ҖлҠ” мӮ¬мҡ©мһҗ мҲҳмҷҖ мһ‘м—… кіөк°„мқҳ мҳҒн–Ҙ, к°ңл°ңмһҗмқҳ лӢӨлЈЁкё° м–ҙл Өмҡҙ м •лҸ„ л“ұ м—¬лҹ¬ мҡ”мқём—җ л”°лқј лӢ¬лқјм§‘лӢҲлӢӨ. к°ҷмқҖ к°ңл°ңмһҗм—җкІҢ лӘҮ лӢ¬м—җ н•ң лІҲм”© нҳёнҷҳм„ұмқ„ кә лҠ” ліҖкІҪ мӮ¬н•ӯмқ„ мІҳлҰ¬н•ҳлҸ„лЎқ мҡ”мІӯн•ҙм„ңлҠ” м•Ҳ лҗ©лӢҲлӢӨ.

лӘЁл“  кІғмқҙ м—¬лҹ¬л¶„мқҙ мӣҗн•  л•Ң н•ӯмғҒ к№Ём§ҖлҠ” кІғмқҖ м•„лӢҷлӢҲлӢӨ. мҳҲмғҒм№ҳ лӘ»н•ң ліҖкІҪ мӮ¬н•ӯкіј лЎӨл°ұн•  мҲҳ м—Ҷмқ„ л•Ң м–ҙл–»кІҢ лҢҖмқ‘н•ҙм•ј н•ҳлҠ”м§Җ (мӮ¬кіјм Ғмқё) м»Өл®ӨлӢҲмјҖмқҙм…ҳ кі„нҡҚмқ„ м„ёмӣҢм•ј н•©лӢҲлӢӨ.

м„Өкі„ н”„лЎңм„ёмҠӨ

мһҳ м •мқҳлҗң API м„Өкі„ к°Җмқҙл“ңлқјмқёмқ„ л„ҳм–ҙм„ң, мҠ¬лһҷмқҳ лӘЁл“  нҢҖмқҙ кіөмҡ© APIлҘј кө¬м¶•н•  мҲҳ мһҲлҠ” м—„кІ©н•ң API м„Өкі„ н”„лЎңм„ёмҠӨлҸ„ мһҲмҠөлӢҲлӢӨ. лӢӨмқҢмқҖ мҡ°лҰ¬к°Җ л”°лҘҙлҠ” лӢЁкі„мһ…лӢҲлӢӨ:

1. API мӮ¬м–‘ мһ‘м„ұн•ҳкё°

мқјлӢЁ нҢҖмқҙ н•ҙкІ°н•ҳкі мһҗ н•ҳлҠ” л¬ём ңлҘј нҷ•мӢӨн•ҳкІҢ н•ҳкі  APIм—җ кҙҖн•ң мӮ¬мҡ© мӮ¬лЎҖлҘј м •мқҳн•ң нӣ„, API мӮ¬м–‘мқ„ мһ‘м„ұн•ҳлҠ” кІғл¶Җн„° мӢңмһ‘н•©лӢҲлӢӨ. мқҙ мӮ¬м–‘мқҖ к°ңл°ңмһҗк°Җ мӮ¬мҡ©н•  APIмқҳ лӢӨм–‘н•ң мёЎл©ҙм—җ кҙҖн•ҙ м„ӨлӘ…н•©лӢҲлӢӨ. м—¬кё°м—җлҠ” л©”м„ңл“ң мқҙлҰ„, лӘ©м Ғ, мҡ”мІӯ мҳҲм ң, мқ‘лӢө л°Ҹ к°ҖлҠҘн•ң м—җлҹ¬мҷҖ к°ҷмқҖ м •ліҙк°Җ нҸ¬н•Ёлҗ  мҲҳ мһҲмңјл©° API м„Өкі„лҘј нҶөн•ҙ м¶©л¶„нһҲ мғқк°Ғн•  мҲҳ мһҲлҠ” кё°нҡҢлҘј мӨҚлӢҲлӢӨ. лҳҗн•ң лӘЁл“  мӮ¬лһҢмқҙ мҡ°лҰ¬мқҳ лӘ©н‘ңк°Җ л¬ҙм—Үмқём§Җ, к·ёлҰ¬кі  мҡ°лҰ¬мқҳ мһҗмӣҗ(resource)мқҙ м–ҙл–»кІҢ л…ём¶ңлҗҳлҠ”м§Җм—җ лҢҖн•ҙ мқјм№ҳн•ҳлҸ„лЎқ н•ҳлҠ” мӨ‘мӢ¬ мҙҲм•Ҳ м—ӯн• мқ„ н•©лӢҲлӢӨ.

к·ёлҹ° кІҪн–Ҙмқҙ мһҲлҠ” нҢҖм—җ мҶҚн•ҙ мһҲлӢӨл©ҙ OpenAPI лҳҗлҠ” AsyncAPIмҷҖ к°ҷмқҖ кіөк°ң нҳ•мӢқмңјлЎң JSON мҠӨнӮӨл§Ҳ л°Ҹ/лҳҗлҠ” мӮ¬м–‘мқ„ мһ‘м„ұн•ҳл©ҙ кө¬нҳ„ м „м—җ API н”„лЎңнҶ нғҖмһ… м ңмһ‘м—җлҸ„ лҸ„мӣҖмқҙ лҗ  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

к°ңл°ңмһҗлҘј мң„н•ҙ мҠӨнӮӨл§ҲлҘј кІҢмӢңн•ҳлҠ” кІғмқҖ мһҗмІҙ нҲҙл§Ғмқ„ мӢңмһ‘н•ҳлҠ” лҚ° лҸ„мӣҖмқҙ лҗ  мҲҳ мһҲм§Җл§Ң, мӮ¬м–‘мқ„ мөңмӢ  мғҒнғңлЎң мң м§Җн•ҳкё° мң„н•ң л…ёл ҘмқҖ мүҪм§Җ м•ҠмҠөлӢҲлӢӨ. мқҙкІғмқҖ мҡ°лҰ¬к°Җ мҠӨмҠӨлЎң к°ңм„ н•  мҲҳ мһҲлҠ” лҳҗ лӢӨлҘё мҳҒм—ӯмһ…лӢҲлӢӨ.

2. лӮҙл¶Җ API лҰ¬л·°

мҪ”л“ңлҘј мһ‘м„ұн•ҳкё° м „м—җ м„Өкі„мғҒмқҳ л¬ём ңлҘј нҢҢм•…н•ҳлҠ” кІғмқҙ нӣЁм”¬ лҚ” нҡЁмңЁм Ғмһ…лӢҲлӢӨ. мҠ¬лһҷм—җм„ңлҠ” API мӮ¬м–‘мқ„ мһ‘м„ұн•ң нӣ„, м—”м§ҖлӢҲм–ҙк°Җ #api-decisionsлқјлҠ” лӮҙл¶Җ мҠ¬лһҷ мұ„л„җм—җм„ң кіөмң н•©лӢҲлӢӨ. мқҙкІғмқҖ APIлҘј кө¬м¶•н•ң м—”м§ҖлӢҲм–ҙк°Җ к°ңл°ң кҙҖкі„, м—”м§ҖлӢҲм–ҙл§Ғ, м ңн’Ҳ кҙҖлҰ¬, к°ңл°ңмһҗ м§Җмӣҗ, нҢҢнҠёл„Ҳ м—”м§ҖлӢҲм–ҙл§Ғ л°Ҹ ліҙм•Ҳ лӢҙлӢ№мңјлЎң кө¬м„ұлҗң м—¬лҹ¬ м§Ғмў…мқҳ к·ёлЈ№м—җм„ң м ңм•Ҳм—җ лҢҖн•ң лӢӨм–‘н•ң кҙҖм җмқ„ м–»мқ„ мҲҳ мһҲлҠ” кё°нҡҢмһ…лӢҲлӢӨ. ліҖкІҪ мӮ¬н•ӯм—җ лҢҖн•ҙ лҚ” к№ҠмқҖ л…јмқҳк°Җ н•„мҡ”н•ң кІҪмҡ° м •кё°м ҒмңјлЎң мҳҲм•Ҫлҗң API м—…л¬ҙ мӢңк°„м—җ кІҖнҶ н•©лӢҲлӢӨ. мқҙ нҡҢмқҳм—җм„ң к·ёлЈ№мқҖ вҖңліҖкІҪ мӮ¬н•ӯмқҙ кё°мЎҙмқҳ лӢӨлҘё APIмҷҖ мқјм№ҳн•©лӢҲк№Ң?вҖқ л°Ҹ вҖңAPI м„Өкі„ к°Җмқҙл“ңлқјмқёкіј мқјм№ҳн•©лӢҲк№Ң?вҖқ мҷҖ к°ҷмқҖ мЈјм ңлҘј л…јмқҳн•©лӢҲлӢӨ. лҳҗн•ң мқҙ к·ёлЈ№мқҖ ліҖкІҪ мӮ¬н•ӯм—җ лҢҖн•ң мқҙлҰ„ м§Җм •, мӮ¬мҡ©м„ұ, ліҙм•Ҳ л°Ҹ м„ұлҠҘ кі л Ө мӮ¬н•ӯм—җ лҢҖн•ҙм„ңлҸ„ мһҗм„ёнһҲ нҶ лЎ н•©лӢҲлӢӨ.

3. мҙҲкё° нҢҢнҠёл„Ҳ н”јл“ңл°ұ

лӮҙл¶Җ мқҙн•ҙлӢ№мӮ¬мһҗмҷҖ н•Ёк»ҳ, лҳҗн•ң APIлҘј кө¬нҳ„н•  мҲҳ мһҲлҠ” мқҙмғҒм Ғмқё к°ңл°ңмһҗмқё нҢҢнҠёл„Ҳм—җкІҢ API мӮ¬м–‘ мҙҲм•Ҳмқ„ м ңкіөн•©лӢҲлӢӨ. мқҙл“Өмқҳ н”јл“ңл°ұмқҖ APIк°Җ н•ҙкІ°н•ҙм•ј н•  л¬ём ңлҘј н•ҙкІ°н•ҳкі  к°ңм„ мқҙ н•„мҡ”н•ң API мёЎл©ҙм—җм„ң мқҙн•ҙн•ҳлҠ” лҚ° мһҲм–ҙ л§Өмҡ° мӨ‘мҡ”н•©лӢҲлӢӨ. мҪ”л“ңлҘј мһ‘м„ұн•ҳкё° м „м—җ мқҙлҹ¬н•ң н”јл“ңл°ұмқ„ л°ӣмқҢмңјлЎңмҚЁ API м„Өкі„лҘј л№ лҘҙкІҢ л°ҳліөн•ҳкі  нӣЁм”¬ лҚ” лӮҳмқҖ APIлҘј кө¬м¶•н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

4. лІ нғҖ н…ҢмҠӨнҠё

мғҲлЎңмҡҙ APIлҘј л„җлҰ¬ мӮ¬мҡ©н•ҳкё° м „м—җ м„ лі„лҗң нҢҢнҠёл„Ҳм—җкІҢ мЎ°кё° м•Ўм„ёмҠӨлҘј м ңкіөн•©лӢҲлӢӨ. мқҙлҹ¬н•ң нҢҢнҠёл„ҲлҠ” мһҗмӮ¬ м ңн’Ҳм—җ APIлҘј нҶөн•©н•ҳкі  APIм—җ лҢҖн•ң мһҗм„ён•ң н”јл“ңл°ұмқ„ м ңкіөн•©лӢҲлӢӨ. мқҙлҘј нҶөн•ҙ API м„Өкі„лҘј лҚ”мҡұ к°ңм„ н•ҳкі  кіөк°ңн•ҳкё° м „м—җ нҷ•мқёлҗң л¬ём ңлҘј н•ҙкІ°н•  кё°нҡҢлҘј м–»мқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

мң м—°м„ұ мң м§Җ

мһҗмӢ л§Ңмқҳ API м„Өкі„ к°Җмқҙл“ңлқјмқёмқ„ м§Ғм ‘ м„Өм •н•ҳкі  APIм—җ лҢҖн•ң кІҖнҶ  н”„лЎңм„ёмҠӨлҘј кө¬м¶•н•ҳлҠ” лҚ° м–ҙл ӨмӣҖмқ„ кІӘм—ҲлӢӨл©ҙ, мқҙм ң нҢҖм—җм„ң к°Җмқҙл“ңлқјмқёмқ„ м–ҙлҠҗ м •лҸ„ лӮҙмһ¬нҷ”н–Ҳмқ„ к°ҖлҠҘм„ұмқҙ нҒҪлӢҲлӢӨ.

м–јл§Ҳ м§ҖлӮҳм§Җ м•Ҡм•„ мҳҲмғҒн–ҲлҚҳ мӢңлӮҳлҰ¬мҳӨмҷҖ мқјм№ҳн•ҳм§Җ м•ҠлҠ” мӮ¬лЎҖк°Җ л°ңмғқн•  кІғмһ…лӢҲлӢӨ. м—¬лҹ¬л¶„мқҖ мһҗмӢ мқҳ мўӢмқҖ мЎ°м–ём—җ м–ҙкёӢлӮҳлҠ” нғҖнҳ‘мқ„ н•  кІғмһ…лӢҲлӢӨ. - л•ҢлЎңлҠ” м—”м§ҖлӢҲм–ҙл§Ғ мЎ°м§Ғм—җ лҸ„мӣҖмқҙ лҗҳкұ°лӮҳ м—¬м „нһҲ мӢӨл§қмҠӨлҹ¬мҡҙ мқјкҙҖм„ұмқ„ м¶”кө¬н•ҳкё° мң„н•ҙ. к·ёлҹ¬лӮҳ л•ҢлЎңлҠ” мҷ„м „нһҲ мғҲлЎңмҡҙ мў…лҘҳмқҳ APIк°Җ м ңкіөлҗҳл©°, к°Җмқҙл“ңлқјмқё м •мӢ м—җм„ң мҳҒк°җ мқ„ л°ӣм•„ м—¬лҹ¬ м§Ғмў…мқҳ мқјмқ„ н•ҳлҠ” нҢҢнҠёл„ҲлҘј м°ЁлӢЁ н•ҙм ңн•  мҲҳ мһҲмҠөлӢҲлӢӨ. м—¬лҹ¬л¶„мқҖ лӮҳмӨ‘м—җ м–ём ңл“ м§Җ м •мұ…мқ„ м •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

л§Ҳм№ҳл©°

м§ҒкҙҖм Ғмқҙкі  мқјкҙҖлҗҳл©° мӮ¬мҡ©н•ҳкё° мү¬мҡҙ APIлҘј м„Өкі„н•ҳлҠ” кІғмқҖ м–ҙл өмҠөлӢҲлӢӨ. мқҙ кёҖм—җм„ңлҠ” мҠ¬лһҷм—җм„ң л”°лҘҙлҠ” API м„Өкі„ мӣҗлҰ¬мҷҖ м„Өкі„ кіјм •мқ„ лӢӨлЈЁм—ҲмҠөлӢҲлӢӨ. APIлҘј кө¬м¶•н•  л•Ң лӘҮ к°Җм§Җ мӨ‘мҡ”н•ң м җмқҖ API м„Өкі„м—җ лҢҖн•ҙ лҜёлҰ¬ мғқк°Ғн•ҳкі  м„Өкі„ м„ нғқмқҙ мқҳлҸ„м Ғмқҙм–ҙм•ј н•ҳл©° м—¬лҹ¬ мқҙн•ҙ кҙҖкі„мһҗмқҳ н”јл“ңл°ұмқ„ лӘЁмңјлҠ” кІғмһ…лӢҲлӢӨ.

мҡ°лҰ¬лҠ” мқҙм ң л§ү мӢңмһ‘н•ҳлҠ” лӢЁкі„мқҙл©°, лӢӨмқҢм—җ л¬ҙм—Үмқ„ н•  мҲҳ мһҲмқ„м§Җ кё°лҢҖлҗ©лӢҲлӢӨ! м—¬лҹ¬л¶„мқҙ к°ңл°ңмһҗк°Җ мўӢм•„н•ҳлҠ” н”Ңлһ«нҸјм—җм„ң мһ‘м—…н•ҳкі  API кө¬м¶•мқ„ мўӢм•„н•ңлӢӨл©ҙ н•Ёк»ҳн•©мӢңлӢӨ!
