Golang for JavaScript developers - Part 2

мӢңмһ‘

л§Ңм•Ҫ лӢ№мӢ мқҙ лӢӨлҘё н”„лЎңк·ёлһҳл°Қ м–ём–ҙлҘј л°°мҡ°лҠ” кІғм—җ лҢҖн•ҙ мғқк°Ғн•ҳкі  мһҲлҠ” Javascript к°ңл°ңмһҗлқјл©ҙ, GolangмқҖ нӣҢлҘӯн•ң м„ нғқмқҙлӢӨ. лӢЁмҲңн•ҳл©° л§ҺмқҖ лӘЁл©ҳн…Җкіј мўӢмқҖ м„ұлҠҘ к·ёлҰ¬кі  JavaScriptмҷҖ мқјл¶Җ мң мӮ¬н•ҳлӢӨ.

мқҙ нҸ¬мҠӨнҠёлҠ” м–ём–ҙлҘј л№„көҗн•ҳлҠ” кІғмқҙ м•„лӢҲкі  к·ёл“Өмқҙ л§Өмҡ° мң мӮ¬н•ҳлӢӨкі  л§җн•ҳкі  мһҲлӢӨ. мқҙкІғмқҖ JavaScript к°ңл°ңмһҗк°Җ Golangмқ„ л№ЁлҰ¬ нҢҢм•…н•  мҲҳ мһҲлҠ” к°Җмқҙл“ңмқҙлӢӨ. Goм—җлҠ” мҡ°лҰ¬к°Җ лӢӨлЈ° мһҗл°”мҠӨнҒ¬лҰҪнҠёмҷҖ мҷ„м „нһҲ лӢӨлҘё л§ҺмқҖ мёЎл©ҙл“Өмқҙ мһҲлӢӨ.

мқҙ мӢңлҰ¬мҰҲмқҳ мқҙм „ нҺём—җм„ң, мҡ°лҰ¬лҠ” JSмҷҖ Go мӮ¬мқҙм—җ мң мӮ¬н•ң кІғл“Өм—җ лҢҖн•ҙ л°°мӣ лӢӨ. л°°мҡҙ кІғл“Ө:

  • Functions
  • мҠӨмҪ”н”„(Scope)
  • нқҗлҰ„ м ңм–ҙ(Flow control)
  • л©”лӘЁлҰ¬ кҙҖлҰ¬(Memory management)

мқҙнҺём—җм„ңлҠ” JSмҷҖ Goмқҳ м°Ёмқҙм җмқ„ лӢӨлЈ° кІғмқҙлӢӨ. л§Ңм•Ҫ мқҙм „ нҺёмқ„ мқҪм–ҙліҙм§Җ м•Ҡм•ҳлӢӨл©ҙ лЁјм Җ мқҪм–ҙліҙмһҗ.

лҚ” лӢӨлҘё кІғл“Ө

ліҙлӢӨмӢңн”ј мқҙ нҺём—җлҠ” мқҙм „ліҙлӢӨ лҚ” л§ҺмқҖ лӮҙмҡ©мқҙ мһҲм§Җл§Ң лӘҮ к°Җм§Җ м°Ёмқҙм җл“ӨмқҖ лҜёл¬ҳн•ҙм„ң JavaScript к°ңл°ңмһҗлҠ” мқҙн•ҙн•ҳкё° мү¬мҡё кІғмқҙлӢӨ.

нғҖмһ… & ліҖмҲҳ(Types & Variables)

мқҙкІғмқҖ мЈјмҡ” м°Ёмқҙм җ мӨ‘ н•ҳлӮҳлӢӨ. JavaScriptлҠ” лҸҷм Ғмқҙкі  лҠҗмҠЁн•ҳкІҢ нғҖмһ…лҗҳкі  GoлҠ” м •м Ғмқҙкі  м—„кІ©н•ҳкІҢ нғҖмһ…лҗңлӢӨ.

JavaScript

var foo = {
    message: "hello"
};

var bar = foo;

// к°ҖліҖ
bar.message = "world";
console.log(foo.message === bar.message); // prints 'true'

// мһ¬н• лӢ№
bar = {
    message: "mars"
};
console.log(foo.message === bar.message); // prints 'false'

Go

var foo = struct {
    message string
}{"hello"}

var bar = foo // fooмқҳ ліөмӮ¬ліёмқ„ л§Ңл“Өкі  barм—җ н• лӢ№н•ңлӢӨ

// barл§Ң ліҖкІҪ
// м°ёкі  bar.messageлҠ” (* bar) .messageмқҳ м•Ҫм–ҙ
bar.message = "world"
fmt.Println(foo.message == bar.message) // prints "false"

// bar мһ¬н• лӢ№
bar = struct {
    message string
}{"mars"}
fmt.Println(foo.message == bar.message) // prints "false"

var barPointer = &foo // fooм—җ нҸ¬мқён„°лҘј н• лӢ№

// fooлҘј ліҖкІҪ
barPointer.message = "world"
fmt.Println(foo.message == barPointer.message) // prints "true"

// fooлҘј мһ¬н• лӢ№
*barPointer = struct {
    message string
}{"mars"}
fmt.Println(foo.message == bar.message) // prints "true"

мң мӮ¬м җ

  • нӮӨмӣҢл“ң varмҷҖ constмқҳ мқҙлҰ„ мҷём—җлҠ” мң мӮ¬м„ұмқҙ м—ҶлӢӨ. Goмқҳ var нӮӨмӣҢл“ңлҠ” лҸҷмһ‘ мёЎл©ҙм—җм„ң JSмқҳ let нӮӨмӣҢл“ңмҷҖ л№„мҠ·н•ҳлӢӨ.
  • var a, foo, bar intкіј к°ҷмқҙ м—¬лҹ¬ varлҘј н•Ёк»ҳ м„ м–ё н•  мҲҳ мһҲлҠ”лҚ° JSмҷҖ мң мӮ¬н•ҳлӢӨ. к·ёлҹ¬лӮҳ Goм—җм„ңлҠ” var a, foo, bar = true, 10, "hello"мҷҖ к°ҷмқҙ лҚ” лӮҳм•„к°Җм„ң мҙҲкё°нҷ” н•  мҲҳ мһҲлӢӨ. JSм—җм„ңлҠ” var [a, foo, bar] = [true, 10, "hello"]мҷҖ к°ҷмқҖ мң мӮ¬н•ң нҡЁкіјм—җ лҢҖн•ҙ кө¬мЎ° л¶„н•ҙ н• лӢ№(destructuring assignment)мқ„ н•  мҲҳ мһҲлӢӨ.

м°Ёмқҙм җ

  • GoлҠ” м»ҙнҢҢмқј мӢң м§Җм •лҗң нғҖмһ…(specified type) лҳҗлҠ” нғҖмһ… м¶”лЎ (type inference)мңјлЎң нғҖмһ… м •ліҙлҘј н•„мҡ”лЎң н•ңлӢӨ.
  • Goм—җлҠ” value нғҖмһ…(primitives, arrays, and structs), reference нғҖмһ…(slice, map & channels) к·ёлҰ¬кі  нҸ¬мқён„°к°Җ мһҲлӢӨ. JSлҠ” value нғҖмһ…(primitives)кіј reference нғҖмһ…(objects, arrays, functions)мқ„ к°Җм§Җкі  мһҲлӢӨ.
  • Goм—җм„ң м„ м–ё н•ң нӣ„ ліҖмҲҳ нғҖмһ…мқ„ ліҖкІҪн•  мҲҳ м—ҶлӢӨ.
  • ліҖмҲҳ н• лӢ№мқҖ Goм—җм„ң лӢЁлқҪ н‘ңнҳ„мӢқ(short-circuit expressions)мқ„ мӮ¬мҡ©н•  мҲҳ м—ҶлӢӨ.
  • varлҠ” Go functionsм—җ : =мҷҖ к°ҷмқҖ к°„лӢЁн•ң л¬ёлІ•мқҙ мһҲлӢӨ.
  • GoлҠ” м—„кІ©н•ҳкІҢ мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” ліҖмҲҳлҘј н—Ҳмҡ©н•ҳм§Җ м•ҠлҠ”лӢӨ. мӮ¬мҡ©лҗҳм§Җ м•ҠмқҖ ліҖмҲҳлҠ” мҳҲм•Ҫ лҗң л¬ёмһҗ мқё _лЎң мқҙлҰ„мқ„ м§Җм •н•ҙм•јн•ңлӢӨ.
  • Goм—җм„ң constлҠ” JavaScriptмҷҖ к°ҷм§Җ м•ҠлӢӨ. Goм—җм„  character, string, boolean лҳҗлҠ” numeric valuesкіј к°ҷмқҖ мӣҗмӢңк°’(primitives)л§Ң м§Җм •н•  мҲҳ мһҲлӢӨ.
  • Goмқҳ ArraysлҠ” кі м • кёёмқҙмқҙлҜҖлЎң JSмҷҖ лӢӨлҘҙлӢӨ. JS ArraysлҠ” лҸҷм ҒмқҙлҜҖлЎң лҸҷм Ғ кёёмқҙлҘј к°Җм§„ arrayмқҳ slices мқё Go slicesмҷҖ лҚ” мң мӮ¬н•ҳлӢӨ.

JavaScript

const foo = ["Rick", "Morty"];

// arrayмқҳ лҒқм—җ м¶”к°Җн•ңлӢӨ.
foo.push("Beth");

// arrayмқҳ лҒқм—җм„ң м ңкұ°н•ңлӢӨ.
element = foo.pop();

Go

foo := []string{"Rick", "Morty"} // sliceлҘј л§Ңл“ лӢӨ

// arrayмқҳ лҒқм—җ м¶”к°Җн•ңлӢӨ.
foo = append(foo, "Beth")

// arrayмқҳ лҒқм—җм„ң м ңкұ°н•ңлӢӨ.
n := len(foo) - 1 // л§Ҳм§Җл§ү мҡ”мҶҢмқҳ index
element := foo[n] // м„ нғқм ҒмңјлЎң л§Ҳм§Җл§ү мҡ”мҶҢлҘј к°Җм ёмҳЁлӢӨ
foo = foo[:n]     // л§Ҳм§Җл§ү мҡ”мҶҢлҘј м ңкұ°н•ңлӢӨ
  • JavaScriptм—җлҠ” dictionariesмҷҖ setsлЎң мӮ¬мҡ©н•  мҲҳмһҲлҠ” Object, Map / Set л°Ҹ WeakMap / WeakSetмқҙ мһҲлӢӨ. Goм—җлҠ” JavaScript ObjectмҷҖ лҚ” мң мӮ¬н•ң лӢЁмҲңн•ң Map л§Ң мқҙлҹ° мқҙмң лЎң к·ё лӘ©м Ғм—җ л¶Җн•©н•ңлӢӨ. лҳҗн•ң Goм—җм„ң mapsлҠ” мҲңм„ңк°Җ м—ҶлӢӨ.(not ordered)

Javascript

const dict = {
    key1: 10,
    key2: "hello"
};

const stringMap = {
    key1: "hello",
    key2: "world"
};

Go

var dict = map[string]interface{}{
    "key1": 10,
    "key2": "hello",
}

var stringMap = map[string]string{
    "key1": "hello",
    "key2": "world",
}

к°ҖліҖм„ұ(Mutability)

JSмҷҖ Goмқҳ лҳҗ лӢӨлҘё мЈјмҡ” м°Ёмқҙм җмқҖ ліҖмҲҳ к°ҖліҖ(variable mutations)мқҙ мІҳлҰ¬лҗҳлҠ” л°©мӢқмқҙлӢӨ. JavaScriptм—җм„ң лӘЁл“  non-primitive ліҖмҲҳлҠ” м°ёмЎ°(reference)лЎң м „лӢ¬лҗңлӢӨ. к·ёлҰ¬кі  к·ё н–үлҸҷмқ„ л°”кҝҖ л°©лІ•мқҖ м—ҶлӢӨ. л°ҳл©ҙм—җ Goм—җм„ңлҠ” slice, map & channelsлҘј м ңмҷён•ң лӘЁл“  кІғмқҙ к°’мңјлЎң м „лӢ¬лҗҳкі  к·ё лҢҖмӢ  нҸ¬мқён„°лҘј ліҖмҲҳм—җ лӘ…мӢңм ҒмңјлЎң м „лӢ¬н•ҳм—¬ мқҙлҘј ліҖкІҪн•ҳлҸ„лЎқ м„ нғқн•  мҲҳ мһҲлӢӨ.

Goм—җм„  мқҙ л•Ңл¬ём—җ JSліҙлӢӨ к°ҖліҖм„ұмқ„ лҚ” мһҳ м ңм–ҙ н•  мҲҳ мһҲлӢӨ.

Javascriptм—җм„ң лҳҗ лӢӨлҘё мЈјлӘ©н• л§Ңн•ң м°Ёмқҙм җмқҖ Goм—җм„ңлҠ” л¶Ҳк°ҖлҠҘн•ң const нӮӨмӣҢл“ңлҘј мӮ¬мҡ©н•ҳм—¬ ліҖмҲҳмқҳ мһ¬н• лӢ№мқ„ л°©м§Җ н•  мҲҳ мһҲлӢӨ.

мң„мқҳ м„№м…ҳм—җм„ң м•Ҫк°„мқҳ ліҖкІҪ к°ҖлҠҘм„ұмқҙ мһҲмқҢмқ„ ліҙм•ҳлӢӨ. мўҖ лҚ” мӮҙнҺҙліҙмһҗ.

JavaScript

let foo = {
    msg: "hello"
};

function mutate(arg) {
    arg.msg = "world";
}
mutate(foo);
console.log(foo.msg); // prints 'world'

Go

type Foo struct {
    msg string
}
var foo = Foo{"hello"}

var tryMutate = func(arg Foo) {
    arg.msg = "world"
}
tryMutate(foo)
fmt.Println(foo.msg) // prints 'hello'

var mutate = func(arg *Foo) {
    arg.msg = "world"
}
mutate(&foo)
fmt.Println(foo.msg) // prints 'world'

м—җлҹ¬ мІҳлҰ¬(Error handling)

м—җлҹ¬ мІҳлҰ¬ мёЎл©ҙм—җм„ң GoмҷҖ JS мӮ¬мқҙмқҳ мң мқјн•ң мң мӮ¬м җмқҖ м—җлҹ¬лҸ„ лӢЁм§Җ к°’ нғҖмһ…(value types)мқҙлқјлҠ” кІғмқҙлӢӨ. л‘җ м–ём–ҙ лӢӨ лӘЁл‘җ м—җлҹ¬лҘј к°’мңјлЎң м „лӢ¬н•  мҲҳ мһҲлӢӨ.

мң„мқҳ м—җлҹ¬ мІҳлҰ¬ мҷём—җлҸ„ л‘җ к°Җм§Җ лӘЁл‘җ мғҒлӢ№нһҲ лӢӨлҘҙлӢӨ. JavaScriptм—җм„ң мҡ°лҰ¬лҠ” л‘ҳ мӨ‘ н•ҳлӮҳлҘј н•  мҲҳ мһҲлӢӨ.

  • try / catch л©”м»ӨлӢҲмҰҳмқ„ мӮ¬мҡ©н•ҳм—¬ async / awaitлҘј м“°лҠ” лҸҷкё° н•ЁмҲҳ(synchronous functions) л°Ҹ л№„лҸҷкё° н•ЁмҲҳ(asynchronous functions)м—җм„ң м—җлҹ¬лҘј нҢҢм•…
  • мҪңл°ұ н•ЁмҲҳ(callback functions)м—җ м „лӢ¬н•ҳкұ°лӮҳ л№„лҸҷкё° н•ЁмҲҳм—җ лҢҖн•ҙ promisesлҘј мӮ¬мҡ©н•ҳм—¬ м—җлҹ¬лҘј мІҳлҰ¬

Goм—җм„ңлҠ” try / catch л©”м»ӨлӢҲмҰҳмқҙ м—Ҷмңјл©° м—җлҹ¬лҘј мІҳлҰ¬н•ҳлҠ” мң мқјн•ң л°©лІ•мқҖ м—җлҹ¬лҘј functionм—җм„ң valueлЎң л°ҳнҷҳн•ҳкұ°лӮҳ panicмңјлЎң мӢӨн–үмқ„ мӨ‘м§Җн•ҳлҠ” кІғмқҙлӢӨ. мқҙкІғмқҖ Goм—җм„ң м—җлҹ¬ мІҳлҰ¬лҘј л§Өмҡ° мһҘнҷ©н•ҳкІҢ л§Ңл“Өкі  Goм—җм„ң мң лӘ…н•ң if err != nil л¬ёмқ„ мһҗмЈј ліҙкІҢн•ңлӢӨ.

JavaScript

function errorCausingFunction() {
    throw Error("Oops");
}

try {
    errorCausingFunction();
} catch (err) {
    console.error(`Error: ${err}`);
} finally {
    console.log(`Done`);
}
// prints
// Error: Error: Oops
// Done

// or the async way

function asyncFn() {
    try {
        errorCausingFunction();
        return Promise.resolve();
    } catch (err) {
        return Promise.reject(err);
    }
}

asyncFn()
    .then(res => console.log(`:)`))
    .catch(err => console.error(`Error: ${err}`))
    .finally(res => console.log(`Done`));
// prints
// Error: Error: Oops
// Done

Go

var errorCausingFunction = func() error {
    return fmt.Errorf("Oops")
}

err := errorCausingFunction()

defer fmt.Println("Done")
// л§Ҳм§Җл§үм—җ к°ҖмһҘ к°Җк№қм§Җл§Ң, enclosing functionмқҳ лҒқлӮ  л•Ңл§Ң мӢӨн–ү
if err != nil {
    fmt.Printf("Error: %s\n", err.Error())
} else {
    fmt.Println(":)")
}
// prints
// Error: Oops
// Done

мғҒмҶҚ лҢҖмӢ  н•©м„ұ(Composition instead of inheritance)

JavaScriptм—җм„ң мғҒмҶҚ(inheritance)мқ„ мӮ¬мҡ©н•ҳм—¬ лҸҷмһ‘мқ„ нҷ•мһҘн•ҳкұ°лӮҳ кіөмң  н•  мҲҳ мһҲмңјл©° GoлҠ” н•©м„ұ(composition)мқ„ лҢҖмӢ  м„ нғқн•ңлӢӨ. JavaScriptм—җлҠ” н”„лЎңнҶ  нғҖмһ… л ҲлІЁ мғҒмҶҚмқҙ мһҲкі  м–ём–ҙмқҳ мң м—°м„ұмңјлЎң мқён•ҙ н•©м„ұмқҙ к°ҖлҠҘн•ҳлӢӨ.

JavaScript

class Animal {
    species;
    constructor(species) {
        this.species = species;
    }
    species() {
        return this.species;
    }
}

class Person extends Animal {
    name;
    constructor(name) {
        super("human");
        this.name = name;
    }
    name() {
        return this.name;
    }
}

var tom = new Person("Tom");

console.log(`${tom.name} is a ${tom.species}`); // prints 'Tom is a human'

Go

type IAnimal interface {
    Species() string
}

type IPerson interface {
    IAnimal
    // IAnimal мқён„°нҺҳмқҙмҠӨ н•©м„ұ
    Name() string
}

type Animal struct {
    species string
}

type Person struct {
    Animal
    // Animal structмқҳ н•©м„ұ
    name   string
}

func (p *Person) Name() string {
    return p.name
}

func (p *Animal) Species() string {
    return p.species
}

func NewPerson(name string) IPerson {
    return &Person{Animal{"human"}, name}
}

func main() {
    var tom IPerson = NewPerson("Tom")
    fmt.Printf("%s is a %s\n", tom.Name(), tom.Species()) // prints 'Tom is a human'
}

лҸҷмӢңм„ұ(Concurrency)

лҸҷмӢңм„ұ(Concurrency)мқҖ Golangмқҳ к°ҖмһҘ мӨ‘мҡ”н•ң кё°лҠҘ мӨ‘ н•ҳлӮҳмқҙл©° л№ӣлӮҳлҠ” л¶Җл¶„мқҙлӢӨ.

JavaScriptлҠ” кё°мҲ м ҒмңјлЎң лӢЁмқј мҠӨл Ҳл“ңмқҙлҜҖлЎң мӢӨм ң кё°ліё лҸҷмӢңм„ұ(real native concurrency)мқҙ м—ҶлӢӨ. service workersмқҳ м¶”к°ҖлҠ” лі‘л ¬ мІҳлҰ¬м—җ лҢҖн•ң м•Ҫк°„мқҳ м§Җмӣҗмқ„ м ңкіөн•ҳм§Җл§Ң м—¬м „нһҲ goroutinesмқҳ м„ұлҠҘкіј лӢЁмҲңм„ұм—җ н•„м Ғн•  мҲҳлҠ” м—ҶлӢӨ. лҸҷмӢңм„ұмқҖ JavaScriptк°Җ нҒ¬кІҢ м§Җмӣҗн•ҳлҠ” л№„лҸҷкё°мӢқ(asynchronous) лҳҗлҠ” л°ҳмқ‘нҳ• н”„лЎңк·ёлһҳл°Қ(reactive programming)кіј к°ҷм§Җ м•ҠлӢӨ.

// мҲңм°Ём Ғ
async function fetchSequential() {
    const a = await fetch("http://google.com/");
    console.log(a.status);
    await a.text();

    const b = await fetch("http://twitter.com/");
    console.log(b.status);
    await b.text();
}

// лҸҷмӢңм Ғмқҙм§Җл§Ң лӢӨмӨ‘ мҠӨл Ҳл“ң м•„лӢҳ
async function fetchConcurrent() {
    const values = await Promise.all([
        fetch("http://google.com/"),
        fetch("http://twitter.com/")
    ]);

    values.forEach(async resp => {
        console.log(resp.status);
        await resp.text();
    });
}

л°ҳл©ҙм—җ GoлҠ” лҸҷмӢңм„ұкіј лі‘л ¬ мІҳлҰ¬м—җ мҷ„м „нһҲ л§һм¶°м ё мһҲлӢӨ. к°ңл…җмқҖ goroutinesкіј channelsлҘј мӮ¬мҡ©н•ҳм—¬ м–ём–ҙм—җ лӮҙмһҘлҗҳм–ҙ мһҲлӢӨ. Goм—җм„ң л№„лҸҷкё° н”„лЎңк·ёлһҳл°Қмқ„ мҲҳн–үн•ҳлҠ” кІғлҸ„ к°ҖлҠҘн•ҳм§Җл§Ң JSмҷҖ лҸҷл“ұн•ң кІғ ліҙлӢӨ лҚ” мһҘнҷ©н•ҳкІҢ ліҙмқёлӢӨ. мқҙкІғмқҖ goroutinesлҘј мӮ¬мҡ©н•ҳм—¬ APIлҘј лҸҷкё°нҷ”лЎң мһ‘м„ұн•ҳкі  л№„лҸҷкё° л°©мӢқмңјлЎң мӮ¬мҡ©н•  мҲҳ мһҲлӢӨлҠ” кІғмқ„ мқҳлҜён•ҳл©° Go м»Өл®ӨлӢҲнӢ°лҠ” мқјл°ҳм ҒмңјлЎң л№„лҸҷкё° API мһ‘м„ұмқ„ л°ҳлҢҖн•ңлӢӨ.

// мҲңм°Ём Ғ
func fetchSequential() {
    respA, _ := http.Get("http://google.com/")
    defer respA.Body.Close()
    fmt.Println(respA.Status)
    respB, _ := http.Get("http://twitter.com/")
    defer respB.Body.Close()
    fmt.Println(respB.Status)
}

// лҸҷмӢң л°Ҹ лӢӨмӨ‘ мҠӨл Ҳл“ң
func fetchConcurrent() {
    resChanA := make(chan *http.Response, 0)

    go func(c chan *http.Response) {
        res, _ := http.Get("http://google.com/")
        c <- res
    }(resChanA)

    respA := <-resChanA
    defer respA.Body.Close()
    fmt.Println(respA.Status)

    resChanB := make(chan *http.Response, 0)

    go func(c chan *http.Response) {
        res, _ := http.Get("http://twitter.com/")
        c <- res
    }(resChanB)

    respB := <-resChanB
    defer respB.Body.Close()
    fmt.Println(respB.Status)
}

м»ҙнҢҢмқј(Compilation)

JavaScriptлҠ” н•ҙм„қмқҙ лҗҳл©°(interpreted) м»ҙнҢҢмқјмқҖ лҗҳм§Җ(compiled) м•ҠлҠ”лӢӨ. мқјл¶Җ JS м—”м§„мқҖ JIT м»ҙнҢҢмқјмқ„ мӮ¬мҡ©н•ҳм§Җл§Ң к°ңл°ңмһҗм—җкІҢлҠ” JavaScriptлҘј мӢӨн–үн•ҳкё° мң„н•ҙ м»ҙнҢҢмқј н•  н•„мҡ”к°Җ м—ҶмңјлҜҖлЎң мӨ‘мҡ”н•ҳм§Җ м•ҠлӢӨ. TypeScript лҳҗлҠ” Babelмқ„ мӮ¬мҡ©н•ң мҪ”л“ң ліҖнҷҳ(Transpiling)мқҖ кі„мӮ°лҗҳм§Җ м•ҠлҠ”лӢӨ рҹҳү

GoлҠ” м»ҙнҢҢмқјлҗҳл©° к·ё л•Ңл¬ём—җ compile-time typeмқҳ м•Ҳм „м„ұкіј м–ҙлҠҗ м •лҸ„мқҳ л©”лӘЁлҰ¬ м•Ҳм „м„ұмқ„ м ңкіөн•ңлӢӨ.

нҢЁлҹ¬лӢӨмһ„

JavaScriptлҠ” мЈјлЎң к°қмІҙ м§Җн–Ҙм Ғмқҙм§Җл§Ң м–ём–ҙмқҳ мң м—°м„ұмңјлЎң л•Ңл¬ём—җ лӘ…л №нҳ•(imperative) лҳҗлҠ” н•ЁмҲҳнҳ•-мҠӨнғҖмқј(functional-style) мҪ”л“ңлҘј мүҪкІҢ мһ‘м„ұн•  мҲҳ мһҲлӢӨ. мқҙ м–ём–ҙлҠ” мғҒлӢ№нһҲ мһҗмң лЎңмҡҙ нҳ•нғңмқҙл©° мӢӨм ңлЎң м•„л¬ҙ кІғлҸ„ к°•мҡ”н•ҳм§Җ м•ҠлҠ”лӢӨ. мқҙкІғмқҖ нҺён–Ҙм Ғ(opinionated)мқҙм§Җ м•Ҡмңјл©° мҰүмӢң мӮ¬мҡ© к°ҖлҠҘн•ң м–ҙл–Ө нӢҖ(tooling)лҸ„ м ңкіөн•ҳм§Җ м•ҠлҠ”лӢӨ. к°ңл°ңмһҗлҠ” к·ё/к·ёл…Җ мһҗмӢ мқҳ нӢҖ(tooling)мқ„ м„Өм •н•  н•„мҡ”к°Җ мһҲмқ„ кІғмқҙлӢӨ.

GoлҠ” кё°ліём ҒмңјлЎң лӘ…л №нҳ•мқҙл©° м•Ҫк°„мқҳ OOP(Object Oriented Programming к°қмІҙ м§Җн–Ҙ н”„лЎңк·ёлһҳл°Қ) л°Ҹ н•ЁмҲҳнҳ•мқ„ мҲҳн–ү н•  мҲҳ мһҲм§Җл§Ң, JavaScriptм—җм„ңмІҳлҹј мүҪм§ҖлҠ” м•ҠлӢӨ. мқҙ м–ём–ҙлҠ” мғҒлӢ№нһҲ м—„кІ©н•ҳкі  нҺён–Ҙм Ғмқҙл©°(opinionated) мҪ”л“ң мҠӨнғҖмқј л°Ҹ нҸ¬л§·нҢ…кіј к°ҷмқҖ кІғмқ„ мӢңн–үн•ңлӢӨ. лҳҗн•ң н…ҢмҠӨнҢ…, нҸ¬л§·нҢ…, л№Ңл”© л“ұмқ„ мң„н•ң лӮҙмһҘ кё°лҠҘлҸ„ м ңкіөн•ңлӢӨ.

кІ°лЎ 

м–ҙл–Ө мӮ¬лһҢмқҙ мӢңлҰ¬мҰҲмқҳ мқҙм „ нҺё лҢ“кёҖм—җм„ң JS к°ңл°ңмһҗк°Җ к°ҖлҠҘн•ң лӘЁл“  мҳөм…ҳ мӨ‘м—җм„ң мҷң GoлҘј м„ нғқн•ҙм•ј н•ҳлҠ”м§Җ л¬јм—ҲлӢӨ. лӮҙ мғқк°Ғм—җ, JSлҠ” мҷ„лІҪн•ң м–ём–ҙк°Җ м•„лӢҲкё° л•Ңл¬ём—җ лӢӨлҘё м–ём–ҙлҘј л°°мҡ°л©ҙ JS к°ңл°ңмһҗк°Җ JSлҘј лҚ” мӢӨмҡ©м ҒмңјлЎң мӮ¬мҡ©н•  мҲҳ мһҲкІҢ нҒ° лҸ„мӣҖмқҙ лҗ  кІғмқҙкі  кё°ліём Ғмқё н”„лЎңк·ёлһҳл°Қ к°ңл…җм—җ лҢҖн•ң к·ёл…Җ/к·ёмқҳ м§ҖмӢқмқ„ к°•нҷ”н•ҳлҠ” лҚ°лҸ„ лҸ„мӣҖмқҙ лҗ  кІғмқҙлӢӨ. л¬јлЎ  Rust, Go, Haskel, Kotlin л“ұкіј к°ҷмқҖ л§ҺмқҖ мҳөм…ҳл“Өмқҙ мһҲм§Җл§Ң, лӮҳлҠ” Goк°Җ мӮ¬мҡ© к°ҖлҠҘн•ң лӘЁл“  мҳөм…ҳ мӨ‘м—җм„ң к°ҖмһҘ лӢЁмҲңн•ҳкі  л„җлҰ¬ мұ„нғқлҗҳм–ҙ мӢңмһ‘н•ҳкё°м—җ мўӢмқҖ кІғмқҙлқјкі  мғқк°Ғн•ңлӢӨ. лӮҳмқҳ л‘җлІҲм§ё м„ нғқмқҖ KotlinмқҙлӮҳ Rustк°Җ лҗ  кІғмқҙлӢӨ.

л§Ҳм№ҳл©°

л¶ҖмЎұн•ң мҳҒм–ҙ мӢӨл ҘмқҙлӮҳл§Ҳ л§ӨлҒ„лҹҪкІҢ лІҲм—ӯмқ„ н•ҳкё° мң„н•ҙ м§Ғм—ӯ л°Ҹ мқҳм—ӯмқ„ лӢӨмҲҳ мӮ¬мҡ©н–ҲлӢӨ.

(л¬ёмһҘмқҳ н•ҙм„қмқҙ мҳӨм—ӯмқҙкұ°лӮҳ мҲҳм •мқҙ н•„мҡ”н•ң м җмқҙ мһҲлӢӨл©ҙ м–ём ңл“  м•Ңл ӨмЈјмӢңл©ҙ л°ҳмҳҒн•ҳкІ мҠөлӢҲлӢӨ.)
